- Finalizado.

-  HTML
-  CSS
-  JAVASCRIPT
-  LEX

#Dependências 

- Linux (testado em: manjaro - localhost)
- gcc 
- gcc-fortran (gfortran)
- flex


#Como usar

- Instalar Dependências
- Copiar projeto do git para servidor
- Executar permissão de execução ao projeto (chmod)
- Abrir página web
